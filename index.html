EYEWARE

An innovative eye-tracking solution leveraging machine learning algorithms and the AdHawk MindFlex pupil tracking glasses to enhance driver attentiveness.

About eyeware

Using the onboard camera of the AdHawk MindLink, we pass video to the openCV machine learning algorithm to classify and detect people (pedestrians) based on their perceived distance from the user and assign them unique IDs. 

Simultaneously, we use the AdHawk Python API to track the user's gaze and cross-reference the detected pedestrians to ensure that the driver has seen the pedestrian.
